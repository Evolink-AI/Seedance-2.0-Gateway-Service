# Disponibilidade Regional

Seedance 2.0 Gateway Service não está disponível nos Estados Unidos, Índia, Rússia e em certas outras regiões devido a restrições do provedor upstream.

Se Seedance 2.0 não estiver disponível na sua região, modelos de vídeo alternativos — incluindo **Kling V3**, **Seedance 1.5 Pro** e **Veo 3.1** — continuam disponíveis através do mesmo EvoLink API Key.

- [Ver todos os modelos disponíveis](https://evolink.ai?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
- [Obter API Key](https://evolink.ai/signup?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
