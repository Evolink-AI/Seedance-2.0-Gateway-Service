# Disponibilité Régionale

Seedance 2.0 Gateway Service n'est pas disponible aux États-Unis, en Inde, en Russie et dans certaines autres régions en raison de restrictions du fournisseur en amont.

Si Seedance 2.0 n'est pas disponible dans votre région, des modèles vidéo alternatifs — notamment **Kling V3**, **Seedance 1.5 Pro** et **Veo 3.1** — restent accessibles via le même EvoLink API Key.

- [Voir tous les modèles disponibles](https://evolink.ai?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
- [Obtenir un API Key](https://evolink.ai/signup?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
