# Региональная доступность

Seedance 2.0 Gateway Service недоступен в Соединённых Штатах, Индии, России и некоторых других регионах из-за ограничений вышестоящего провайдера.

Если Seedance 2.0 недоступен в вашем регионе, альтернативные модели для генерации видео — включая **Kling V3**, **Seedance 1.5 Pro** и **Veo 3.1** — остаются доступными через тот же EvoLink API Key.

- [Получить API Key](https://evolink.ai/signup?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
