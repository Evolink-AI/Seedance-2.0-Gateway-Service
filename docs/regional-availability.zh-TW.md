# 區域可用性

由於上游服務商限制，Seedance 2.0 Gateway Service 在美國、印度、俄羅斯及部分其他地區不可用。

如果 Seedance 2.0 在您所在地區不可用，替代影片模型（包括 **Kling V3**、**Seedance 1.5 Pro** 和 **Veo 3.1**）仍可透過同一個 EvoLink API Key 使用。

- [查看所有可用模型](https://evolink.ai?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
- [取得 API Key](https://evolink.ai/signup?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
