# 区域可用性

由于上游服务商限制，Seedance 2.0 Gateway Service 在美国、印度、俄罗斯及部分其他地区不可用。

如果 Seedance 2.0 在您所在地区不可用，替代视频模型（包括 **Kling V3**、**Seedance 1.5 Pro** 和 **Veo 3.1**）仍可通过同一个 EvoLink API Key 使用。

- [查看所有可用模型](https://evolink.ai?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
- [获取 API Key](https://evolink.ai/signup?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
