# 地域別提供状況

Seedance 2.0 Gateway Service は、上流プロバイダーの制限により、アメリカ合衆国、インド、ロシア、およびその他一部の地域ではご利用いただけません。

お住まいの地域で Seedance 2.0 が利用できない場合でも、**Kling V3**、**Seedance 1.5 Pro**、**Veo 3.1** などの代替ビデオモデルは、同じ EvoLink API Key でご利用いただけます。

- [利用可能な全モデルを見る](https://evolink.ai?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
- [API Key を取得](https://evolink.ai/signup?utm_source=github&utm_medium=readme&utm_campaign=seedance-2-api)
